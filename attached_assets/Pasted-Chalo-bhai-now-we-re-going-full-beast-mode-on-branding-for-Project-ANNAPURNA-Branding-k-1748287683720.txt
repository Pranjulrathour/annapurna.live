## 3. ✍️ Typography System

### 🅰️ Primary Font – **"Poppins"**

* Modern, legible, open, and perfect for mobile and web
* Font Weights: 300 (Light), 400 (Regular), 600 (Semi-Bold), 700 (Bold)

### 🅱️ Secondary Font – **"Inter"**

* Use for dashboards, data, buttons & smaller UI text
* Font Weights: 400, 500, 700

---

### Typography Rules

| Usage             | Font    | Weight | Size Example     | Usage Place                            |
| ----------------- | ------- | ------ | ---------------- | -------------------------------------- |
| Main Headers      | Poppins | 700    | 28–36px (xl-2xl) | Homepage, Hero section, Page titles    |
| Subheadings       | Poppins | 600    | 20–24px (lg-xl)  | Section headers, Dashboard Cards       |
| Body Text         | Inter   | 400    | 14–16px          | Donation info, Card description, Forms |
| Button Text       | Inter   | 600    | 14–16px          | CTA Buttons like "Donate Now", "Claim" |
| Captions / Labels | Inter   | 400    | 12px             | Tags, Timestamp, Footnotes             |
